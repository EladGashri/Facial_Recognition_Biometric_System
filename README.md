Industry project done in collaboration with Amdocs. Project designed as a facial recognition attendance system for a company with multiple branches and thousands of employees.

Multithreaded REST API written with Java and Spring Framework for detailed attendance reports and for editing and adding employees. Database in MongoDB.

Multithreaded live feed that registers employees' attendance from multiple cameras in different locations simultaneously. Written with Java.

Neural network facial recognition algorithm written with Python and PyTorch.
